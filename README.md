# Dynamic Background Color Changer 🌈

A fun, interactive mini-project that dynamically changes the website's background color. This is a great demonstration of DOM manipulation and event handling in JavaScript.

## 🌟 Key Features
- **Random Color Generator:** Click a button to generate a completely random HEX color code.
- **Color Display:** The screen displays the exact HEX code of the current background color.
- **Smooth Transitions:** CSS transitions make the color change smooth and pleasing to the eye.
- **Copy to Clipboard:** (Optional) Click the HEX text to copy the color code for your own use.

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **Vanilla JavaScript**

## 🚀 How to Run
1. Clone the repo and open `index.html` in your browser.
2. Click the specific buttons on the screen to watch the colors change!

## 🧠 What I Learned
I improved my understanding of JavaScript Math functions (`Math.random()`), manipulating inline CSS styles via JS, and structuring clean event listeners.
